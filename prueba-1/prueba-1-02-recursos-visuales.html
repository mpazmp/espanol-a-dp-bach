Practica

Tres ejercicios de dificultad creciente para afianzar el análisis visual.

Ejercicio 1 · Identifica

Nombra el recurso visual

Lee la descripción de cada elemento visual y elige el nombre del recurso que representa. Hay una sola respuesta correcta en cada caso.

Ejercicio 2 · Distingue

¿Cuál es el efecto más preciso?

Para cada elemento visual ya identificado, elige el efecto más preciso que produce en el receptor.

Ejercicio 3 · Analiza

Completa la fórmula

Aplica la fórmula completa al recurso indicado eligiendo la opción más precisa en cada paso.
