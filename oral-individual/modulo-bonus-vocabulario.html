📈 De correcto a excelente

Sustituir estas palabras es el cambio más rápido para subir el nivel del Criterio D.

❌ Nivel 5–6 · Plano
  • muestra / refleja
  • hay una metáfora / usa la metáfora
  • esto es importante porque
  • el autor quiere decir
  • se puede ver que
  • tiene mucho significado
  • el lector siente
  • esto nos hace pensar en
  • es una forma de expresar
  • el tema de... está presente
✦ Nivel 7–10 · Preciso
  • construye / articula / materializa
  • a través de esta metáfora, el autor logra...
  • esto es significativo en tanto que
  • la decisión del autor de... produce el efecto de
  • el texto hace visible / revela / problematiza
  • esta elección léxica es determinante porque
  • el lector es interpelado / posicionado
  • esto activa / sugiere / pone en tensión
  • funciona como / opera como / actúa como
  • la cuestión global se densifica / emerge
